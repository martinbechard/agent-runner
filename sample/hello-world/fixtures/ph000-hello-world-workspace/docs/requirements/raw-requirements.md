# Request: Hello World Python App

Build a very small Python application that prints `Hello, world!` to
standard output.

Requirements:

- Use Python 3.
- The app should be runnable from the command line.
- Keep the implementation intentionally minimal.
- Include a short README with run instructions.
- Include one automated test that verifies the program prints the
  expected output.

Constraints:

- Do not introduce unnecessary frameworks.
- Prefer a simple file layout that is easy to understand.

Definition of done:

- A human can run the app from the command line and see `Hello, world!`.
- A human can run the test suite and see the test pass.
